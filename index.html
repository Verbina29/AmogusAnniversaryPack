Installing the Modpack

  1. Install PrismLauncher.
  2. If you do not already have it, install Java 17.
  3. After following the prompts to set up PrismLauncher, grab the instance file.
  4. Press "Add Instance" in PrismLauncher.
  5. Select "Import from zip".
  6. Select "Browse", and locate the instance file.
  7. Press "OK".
  8. Launch the installed instance.

This instance should automatically update whenever I update the modpack. If you have any issues installing the pack, joining the server, or just want to chat, join the Discord server.
